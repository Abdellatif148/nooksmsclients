Nook SMS
API Documentation

La première infrastructure SMS moderne construite pour les développeurs marocains. Envoyez des SMS transactionnels, OTP et promotionnels en quelques lignes de code, avec des prix en dirhams.

Production: https://api-nook.up.railway.app
Sandbox: https://api-nook.up.railway.app/sandbox

🚀 Démarrage rapide

Envoyez votre premier SMS en moins de 5 minutes.

1

Créez un compte

Inscrivez-vous et obtenez votre clé API instantanément.

2

Envoyez votre premier SMS

// Node.js — npm install node-fetch
const response = await fetch('https://api-nook.up.railway.app/v1/sms/send', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer nk_live_VOTRE_CLE_API',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    to: '+212612345678',
    message: 'Votre code: 4829. Valide 10 min.'
  })
})
const data = await response.json()
// { success: true, message_id: 'nk_xxx', cost: 0.20 }

🔑 Authentification

Toutes les requêtes nécessitent votre clé API dans le header Authorization.

Authorization: Bearer nk_live_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
⚠️ Ne partagez jamais votre clé API. Stockez-la dans vos variables d'environnement (process.env.NOOK_API_KEY), jamais dans votre code source ou votre dépôt Git.

🧪 Mode Sandbox

📦 Envoi en masse

POST/v1/sms/bulkEnvoyer à plusieurs numéros
// Campagne promotionnelle — ex: Hmizate
const { data } = await axios.post('/v1/sms/bulk', {
  recipients: ['+212612345678', '+212698765432', '+212670123456'],
  message: `🔥 FLASH SALE!\n-50% sur l'électro CE SOIR seulement\nJusqu'à minuit: hmizate.ma/flash`,
  sender_id: 'HMIZATE',
  message_type: 'promotional'
}, { headers })

// Réponse immédiate avec job_id
// { accepted: true, job_id: 'job_xxx', total: 3, queued: 3 }

// Suivre la progression
const status = await axios.get(`/v1/sms/bulk/${data.job_id}`, { headers })
// { progress: 85, sent: 255, failed: 2, total: 300 }
response = requests.post(
    '/v1/sms/bulk',
    headers=headers,
    json={
        'recipients': ['+212612345678', '+212698765432'],
        'message': 'Votre message ici',
        'message_type': 'promotional'
    }
)
ℹ️ Les envois en masse sont traités par une file d'attente Redis. Vous recevez immédiatement un job_id pour suivre la progression. Le webhook est déclenché à la fin.

🔐 OTP — Codes de vérification

Envoyez et vérifiez des codes OTP sécurisés en 2 étapes. Les codes sont générés côté serveur, hashés, et expirés automatiquement après 10 minutes.

POST/v1/otp/sendGénérer et envoyer un OTP
POST/v1/otp/verifyVérifier un code OTP
// ÉTAPE 1: Envoyer l'OTP
const { data: otpData } = await axios.post('/v1/otp/send', {
  to: '+212661234567',
  purpose: 'login',  // login | verification | payment
  language: 'fr'         // fr | ar | en
}, { headers })

// SMS reçu: "Code de connexion Nook: 482910. Valide 10 min."
// Sauvegarder otp_id en session
const otpId = otpData.otp_id

// ÉTAPE 2: Vérifier le code entré par l'utilisateur
const { data: result } = await axios.post('/v1/otp/verify', {
  otp_id: otpId,
  phone: '+212661234567',
  code: '482910'
}, { headers })

if (result.valid) {
  // ✓ Utilisateur authentifié — créer la session
  createUserSession(result.phone)
}
# Étape 1: Envoyer
otp = requests.post('/v1/otp/send',headers=h,json={
    'to':'+212661234567','purpose':'login','language':'fr'
}).json()

# Étape 2: Vérifier
result = requests.post('/v1/otp/verify',headers=h,json={
    'otp_id': otp['otp_id'],
    'phone': '+212661234567',
    'code': user_input
}).json()
if result['valid']:
    create_session()

🔔 Webhooks

Recevez des notifications en temps réel quand le statut d'un message change. Configurez votre URL dans votre compte.

Événements disponibles

ÉvénementDéclenché quand
message.deliveredSMS livré au destinataire
message.failedÉchec d'envoi (numéro invalide, etc.)
bulk.completedCampagne en masse terminée
credits.lowCrédits passent sous le seuil configuré
otp.verifiedCode OTP vérifié avec succès

Payload webhook

{
  "event": "message.delivered",
  "timestamp": "2026-07-01T10:23:44.000Z",
  "data": {
    "message_id": "nk_9032844707704de1a0b7",
    "to": "+212642738171",
    "status": "delivered",
    "network": "maroc_telecom",
    "cost": 0.20
  }
}
⚠️ Vérifiez toujours la signature webhook avec le header X-Nook-Signature pour éviter les requêtes frauduleuses.
