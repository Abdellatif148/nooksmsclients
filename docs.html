Nook SMS API

La première infrastructure SMS moderne construite pour les développeurs marocains. Envoyez des SMS transactionnels, OTP et promotionnels en quelques lignes de code.

Base URL: https://api.nook.ma
Couverture: 🇲🇦 Maroc • Maroc Telecom • Inwi • Orange

🚀 Démarrage rapide

Envoyez votre premier SMS en moins de 5 minutes.

1

Créer un compte

Inscrivez-vous et obtenez votre clé API instantanément.

POST /v1/clients/register
2

Installer la librairie

# Node.js
npm install axios

# Python
pip install requests
3

Envoyer votre premier SMS

const response = await fetch('https://api.nook.ma/v1/sms/send', {

🔑 Authentification

Toutes les requêtes nécessitent votre clé API dans le header Authorization.

Authorization: Bearer nk_live_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
⚠️ Ne partagez jamais votre clé API. Stockez-la dans vos variables d'environnement, jamais dans votre code source.

📤 Envoyer un SMS

POST /v1/sms/send Envoyer un SMS à un numéro

Paramètres

ParamètreTypeRequisDescription
tostringRequisNuméro marocain: +212XXXXXXXXX ou 0XXXXXXXXX
messagestringRequisContenu du message (max 160 caractères)
sender_idstringOptionnelNom affiché (ex: MONAPP) — nécessite enregistrement
message_typestringOptionneltransactional | promotional | otp (défaut: transactional)

Exemple — OTP pour une app de livraison

// Exemple: Rapidio Delivery — vérification à la livraison
const { data } = await axios.post('https://api.nook.ma/v1/sms/send', {
  to: '+212642738171',  // Numéro du client
  message: `Votre livraison Rapidio est arrivée!\nCode de confirmation: 7392\nValable 10 minutes.`,
  sender_id: 'RAPIDIO'
}, {
  headers: { Authorization: `Bearer ${process.env.NOOK_API_KEY}` }
})

console.log(data)
// { success: true, message_id: 'nk_xxx', cost: 0.20, ... }

🔐 OTP — Codes de vérification

Envoyez et vérifiez des codes OTP sécurisés. Les codes sont générés, hashés, et expirés automatiquement.

POST /v1/otp/send Générer et envoyer un code OTP 
// Étape 1: Envoyer l'OTP au numéro du client
const { data } = await axios.post('/v1/otp/send', {
  to: '+212661234567',
  purpose: 'login',  // login | verification | payment
  language: 'fr'        // fr | ar
}, { headers })

// Stocker l'otp_id pour la vérification
const otpId = data.otp_id

// SMS reçu: "Code de connexion: 482910. Valide 10 min."

// Étape 2: Vérifier le code entré par l'utilisateur
const verify = await axios.post('/v1/otp/verify', {
  otp_id: otpId,
  phone: '+212661234567',
  code: '482910'  // Code saisi par l'utilisateur
}, { headers })

if (verify.data.valid) {
  // ✓ Authentification réussie
}

🇲🇦 Cas d'usage marocains

Exemples concrets pour les types d'entreprises au Maroc.

🚗 Application de livraison (ex: Glovo Maroc)

// 1. Confirmation de commande
await sendSMS(client.phone, `Commande #${order.id} confirmée ✓\nEstimation: ${order.eta} min\nTotal: ${order.total} DH`)

// 2. Livreur en route
await sendSMS(client.phone, `Votre livreur est en route 🛵\nCode de confirmation: ${order.code}`)

// 3. Livraison effectuée
await sendSMS(client.phone, `Livraison effectuée ✓\nMerci de votre confiance!\nNotez votre livreur sur l'app.`)

🏦 Application bancaire / CIH / Attijariwafa

// Virement reçu
await sendSMS(client.phone, `CIH Bank: Virement reçu de 2,500.00 MAD\nSolde: 8,750.00 MAD\nRéf: ${ref}`)

// OTP pour connexion sécurisée
const otp = await sendOTP({
  to: client.phone,
  purpose: 'login',
  language: 'ar'  // رمز بالعربية
})
